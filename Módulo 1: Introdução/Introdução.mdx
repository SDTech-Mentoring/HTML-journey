# Módulo 1: Introdução ao HTML

## Tópicos:

- O que é HTML e sua importância?
- Estrutura básica de um documento HTML
- Tags e elementos fundamentais
- Atributos de elementos HTML

### Atividades Práticas:

1. Explique o que é HTML e por que é importante para a web.
2. Crie um documento HTML simples com título e parágrafos, me contando um pouco sobre você.
3. Experimente adicionar diferentes atributos aos elementos HTML para personalizar ainda mais sua página. Você pode incluir informações adicionais sobre você, como hobbies, cidade e muito mais.
